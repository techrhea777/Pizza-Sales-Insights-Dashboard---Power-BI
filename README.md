 **Pizza Sales Insights Dashboard - Power BI**
 
**Project Overview**
This project analyzes pizza sales data to uncover key business insights using SQL and Power BI.
The goal is to evaluate sales performance, identify top and bottom products, and understand customer ordering patterns.

**Dataset**
File: pizza_sales.csv
Records: ~400 rows
Contains information on:
Order ID
Pizza Name
Category & Size
Quantity
Price
Order Date

**Tools & Technologies**
SQL – for data querying and KPI calculations
Power BI – for data visualization and dashboard creation

**Key KPIs Analyzed**

The following KPIs were calculated using SQL:
💰 Total Revenue
🧾 Total Orders
🍕 Total Pizzas Sold
📈 Average Order Value
📊 Average Pizzas per Order
📈 Analysis Performed

🔹 Sales Trends
Daily order trends
Monthly order trends
Peak ordering days and times

🔹 Product Performance
Top 5 pizzas by:
Revenue
Quantity
Total Orders
Bottom 5 pizzas by:
Revenue
Quantity
Total Orders

🔹 Category Insights
Sales distribution by pizza category
Sales distribution by pizza size

**Key Insights**
Classic category contributes the highest sales and orders
Large size pizzas generate the most revenue
Peak orders occur on weekends, especially evenings
Certain pizzas consistently underperform and may need review
